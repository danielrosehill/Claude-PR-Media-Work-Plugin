# Analyze Performance

Analyze communications metrics and generate actionable insights.

## Prerequisites

Read:
- `context/strategy/goals-and-objectives.md`
- `context/historical/performance-benchmarks.md`
- Performance data in `inputs/metrics/`

## Analysis Setup

Gather from user:
1. **Analysis scope**: Overall, campaign-specific, channel-specific
2. **Time period**: What dates to analyze
3. **Data available**: What metrics data exists
4. **Comparison**: Compare to what (prior period, benchmark, goal)
5. **Focus areas**: Any specific questions to answer

## Analysis Process

Use the `metrics-analyst` agent for detailed analysis.

### Steps:
1. **Data review**: Understand available data
2. **Performance assessment**: Compare to objectives/benchmarks
3. **Trend analysis**: Identify patterns
4. **Insight development**: Answer "so what?"
5. **Recommendations**: Actionable next steps

### Key Questions:
- Are we on track to meet objectives?
- What's working and what isn't?
- Where are the biggest opportunities?
- What should we do differently?

## Report Types

### Monthly Performance Report
- KPI performance vs. targets
- Channel performance summary
- Key insights and learnings
- Recommendations

### Campaign Performance Report
- Campaign results vs. objectives
- Channel breakdown
- What worked/didn't work
- Recommendations for future

### Quick Analysis
- Answer specific questions
- Spot-check performance
- Identify issues or opportunities

## Output

Generate appropriate report format based on scope.

Save to: `outputs/reports/[date]-[report-type].md`

Update benchmarks: `context/historical/performance-benchmarks.md`

## Completion

1. Present key findings
2. Highlight actionable recommendations
3. Suggest follow-up analyses if needed
4. Commit report
