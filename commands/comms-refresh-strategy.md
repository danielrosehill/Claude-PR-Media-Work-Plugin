# Refresh Communications Strategy

Review the current communications strategy and update it based on evolving objectives, performance data, and market conditions.

## Prerequisites

Read these context files:
- `context/strategy/communications-strategy.md`
- `context/strategy/goals-and-objectives.md`
- `context/historical/performance-benchmarks.md`
- `context/historical/campaign-history.md`
- Any recent reports in `outputs/reports/`

## Review Process

### 1. Current State Assessment

Review and discuss:
- What were the original strategic objectives?
- What has been accomplished so far?
- What has changed since strategy was established?
  - Business objectives
  - Market conditions
  - Competitive landscape
  - Audience behavior
  - Resource availability

### 2. Performance Review

Analyze:
- Performance against KPIs
- What's working well
- What's underperforming
- Unexpected results or learnings

### 3. Gap Analysis

Identify:
- Gaps between current and desired performance
- Missing capabilities or resources
- Opportunities not being pursued
- Threats not being addressed

### 4. Strategy Refresh

Update strategy to:
- Reflect current objectives
- Address performance gaps
- Capitalize on learnings
- Adapt to changed conditions

### 5. Action Planning

Define:
- Specific changes to tactics
- Resource reallocation
- New initiatives to launch
- Activities to discontinue

## Output

Update:
- `context/strategy/communications-strategy.md`
- `context/strategy/goals-and-objectives.md` (if objectives changed)
- `context/strategy/key-messages.md` (if messaging evolved)

Create:
- Strategy refresh summary in `outputs/strategy/[date]-strategy-refresh.md`

## Completion

1. Summarize key changes to strategy
2. Outline immediate next steps
3. Note any decisions needed from stakeholders
4. Commit changes
